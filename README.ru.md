# 🏋️‍♂️ FitPulse

**FitPulse** это приложение для отслеживания фитнес-тренировок на базе искусственного интеллекта, которое использует компьютерное зрение для обнаружения отжиманий в реальном времени, подсчёта повторений, отслеживания темпа тренировки, сохранения истории занятий в Excel и построения графиков производительности.

![FitPulseDemo](https://github.com/KrishBharadwaj5678/FitPulse/raw/main/FitPulseDemo.gif)

## 🚀 Возможности

| Функция                                          | Описание                                                  |
| ------------------------------------------------ | --------------------------------------------------------- |
| 🎯 Обнаружение отжиманий в реальном времени      | Определяет движения при отжиманиях с помощью анализа позы |
| 🔢 Автоматический подсчёт повторений             | Точно считает выполненные отжимания                       |
| ⏱️ Отслеживание темпа тренировки                 | Измеряет время выполнения каждого отжимания               |
| ⚠️ Система предупреждения о неправильной технике | Уведомляет пользователя о неправильном положении тела     |
| 📈 Индикатор прогресса                           | Показывает прогресс по достижению цели тренировки         |
| 🔊 Звуковая обратная связь                       | Воспроизводит звук после каждого завершённого повторения  |
| 📄 Запись данных в Excel                         | Автоматически сохраняет историю тренировок                |
| 📊 Графики производительности                    | Отображает аналитику тренировки после завершения занятия  |
| 📷 Отслеживание через камеру                     | Анализирует упражнения напрямую через веб-камеру          |
| 📹 Поддержка видеофайлов                         | Анализирует ранее записанные видеоролики                  |
| 🖼️ Экспорт графиков в PNG                       | Сохраняет графики в виде изображений                      |
| 🎨 Чистый интерфейс                              | Простой и удобный пользовательский интерфейс              |

## ⚙️ Установка

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/FitPulse.git
```

### 2️⃣ Перейти в директорию проекта

```bash
cd FitPulse
```

### 3️⃣ Установить зависимости

```bash
pip install -r requirements.txt
```

## 4️⃣ Запуск приложения

### 📹 Режим анализа видео

```bash
python main.py
```

#### ИЛИ

### 📷 Режим работы с веб-камерой
